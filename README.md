# 쇼핑 리스트 앱

로컬스토리지 기반 쇼핑 리스트 웹 앱입니다.

## 기능

- 아이템 추가 (버튼 클릭 또는 Enter 키)
- 체크박스로 완료/미완료 토글
- 전체 / 미완료 / 완료 필터링
- 개별 아이템 삭제
- 완료 항목 일괄 삭제
- localStorage에 데이터 자동 저장 (새로고침 후에도 유지)

## 사용법

`shopping-list.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

## 테스트

Playwright MCP를 사용한 12개 테스트 케이스 전원 통과.  
자세한 결과는 `test-report.md` 참고.
